ABOUT PROJECT

“Люди считают, что программирование — это наука избранных, но в реальности все наоборот — просто много людей создают программы, которые используют чужие программы, как-будто строя стену из маленьких кирпичиков.”

Donald Knuth

Данный проект является серьёзным испытанием для меня. Это одна из первых серьезный работ, которую готов показать широкой публике.

Программа мониторит разные банки, затем достает у них поточный курс валют, который можно просматривать в режиме реального времени. Обновления курса идет регулярно.

Так же проектом предусмотрены графики изменения валют по каждому банку и средней валюты по банкам. Есть возможность просмотреть историю колебания валюты за определенный период или курс за определенную дату.

Проект допиливается изо дня в день. Спасибо что оценили :)
